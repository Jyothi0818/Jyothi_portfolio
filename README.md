Personal Portfolio Website Using HTML & CSS

Developed a responsive personal portfolio website using HTML5 and CSS3 to showcase my academic profile, technical skills, certifications, education, hobbies, and contact details. The website features a clean and modern user interface with a responsive layout, navigation bar, hero section, skills section, education timeline, certifications, and contact information. The project demonstrates my front-end web development skills, including semantic HTML, CSS Flexbox, responsive design, and user-friendly styling. It serves as a professional online profile to present my achievements and technical expertise.

HTML5
CSS3

Key Features:

Responsive and modern design
Navigation bar with smooth section links
Hero/About section
Education details
Skills section
Certifications section
Hobbies and Languages
Contact information
Clean and attractive user interface

Objective:
To create a professional portfolio website that highlights my educational background, technical skills, and achievements while improving my front-end web development skills.
